Mission

This website aims to effectively spread awareness toward creating a more eco-friendly internet whilst itself remaining eco-friendly!

Carbon

According to Website Carbon, this web page achieves a carbon rating of A+, which is cleaner than 100% of all web pages globally! Click the link to test other web pages for their carbon cleanliness!

Sustainable Features of This Site

  • System Fonts: Uses fonts already on your device to save 100kb+ of data.
  • No Frameworks: Built with basic HTML/CSS to reduce server processing.
  • Earth-Toned Palette: Designed for high readability and lower screen brightness.
  • Static Media: Uses a static image instead of an animated GIF to eliminate continuous CPU/GPU processing energy.
